Oracle Smart View Installation Guide

Step 1: Download & Install Oracle Smart View

Step 1.1: Download Oracle Smart View

Option 1: Use Oracle Link

  1. Navigate to the Financial Reporting Center and select the Open Workspace for Financial Reports task.
  2. In the Enterprise Performance Management System Workspace, select Tools > Install > Smart View.

Option 2: Use Search Engine

In your favourite search engine, type "Oracle Smart View for Office Downloads" and click on the Oracle Smart View for Office Downloads link. Follow the instructions.

Step 1.2: Install Oracle Smart View

  1. The Oracle Download Manager tool will be downloaded initially. Using the tool, download the Smart View installation executable.
  2. Double-click on the Smart View executable to install the plugin software.

Step 2: Enable Oracle Smart View Plugin

  1. Once the software is successfully installed, open the Microsoft Excel application and navigate to File > Options > Add-ins.
  2. Under the Manage section, select COM Add-ins and click the Go button.
  3. Select Oracle Smart View for Office and click OK.

This step will enable the Smart View Plugin for Excel.

Step 3: Connect to Fusion Applications

  1. Open Excel. Navigate to the Smart View menu > Options.
  2. Click the Options button and select the Advanced option.
  3. Paste the URL in the Shared Connections URL field. Click OK.
  4. Navigate to Panel > Shared Connections and the Oracle Applications window will open. Sign in to Fusion Applications.
  5. After successful login, navigate to Shared Connections > Select > Oracle Essbase > Under Essbase Cluster > Select Chart of Accounts Name specific to your organization.
  6. Click Connect.
